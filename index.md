# High-fidelity CFD for turbomachinery, aerodynamics and acoustics

Zethesis Research & Engineering provides computational fluid dynamics consulting and simulation studies for engineering teams — using RANS where it fits, and high-order LES when the physics demands it.

### What we do

We run the CFD studies engineering teams need to understand and improve flow-driven designs — and, where it matters, we build the software and automation that make those results repeatable, scalable on HPC, and defensible.

### Where we focus

- **Computational fluid dynamics** — solver setup, meshing, turbulence modelling and validation
- **Scientific software** — turning prototypes and research code into maintainable tools
- **HPC and GPU computing** — parallel and accelerated implementations for demanding cases
- **Workflow automation** — repeatable pipelines for simulation, validation and post-processing

Typical applications include axial and radial turbomachinery, hydraulic turbines, external aerodynamics, aeroacoustics and hydroacoustics.
